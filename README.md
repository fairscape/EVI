# EVI: Evidence Graph Ontology
The Evidence Graph ontology extends core concepts from the W3C Provenance Ontology <a href="https://www.w3.org/TR/prov-o/">PROV-O</a>, <a href="https://schema.org/">Schema.org</a>, and <a href="https://bioschemas.org/profiles/">Bioschemas' Profiles</a> to describe evidence for correctness of findings in biomedical publications. The semantic data model in EVI is expressed using OWL2 Web Ontology Language.
